# 高级用法

## 自定义规则

### 规则语法
学习如何使用模式匹配创建自定义屏蔽规则：

```javascript
{
  "pattern": "联系人名称",
  "action": "block",
  "type": "message"
}
```

### 规则类型
- 基于联系人的规则
- 模式匹配规则
- 正则表达式规则

## 故障排除

### 常见问题
- 插件无法工作
- 规则不生效
- 配置问题

### 解决方案
- 重启插件
- 重置插件设置
- 更新屏蔽规则
