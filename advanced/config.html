高级配置指南 ​

配置文件结构 ⚙️ ​

配置文件采用JSON格式，包含以下主要部分：

json
{
  "blockConfig": {
    "exactMatch": [],
    "includesMatch": [],
    "userSpecific": {},
    "superEmoji": true
  }
}

配置项说明 📋 ​

exactMatch ​

精确匹配规则列表，完全匹配才触发屏蔽。

includesMatch ​

包含匹配规则列表，包含关键词即触发屏蔽。

userSpecific ​

针对特定用户的屏蔽规则。

superEmoji ​

控制是否启用超级表情屏蔽。

配置导入导出 💾 ​

导出配置 ​

  1. 点击设置面板中的"导出配置"
  2. 选择保存位置
  3. 配置将被保存为JSON文件

导入配置 ​

  1. 点击"导入配置"
  2. 选择之前导出的配置文件
  3. 确认导入即可

进阶技巧 🚀 ​

配置合并 ​

可以合并多个配置文件，实现规则共享。

规则优化 ​

  • 定期清理重复规则
  • 合并相似规则
  • 优化匹配顺序

备份策略 ​

建议：

  • 定期备份
  • 保留多个版本
  • 使用版本控制工具
